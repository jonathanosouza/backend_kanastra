# kanastra - Configuração do Backend

<!-- PROJECT -->

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jonathanosouza/backend_kanastra.git
   ```
2. Start the Docker
   ```sh
   docker-compose up -d
   ```
3. Install the dependences:
    ```sh
    python install_dependencies.py
    ```
### Migrations Database 

4. Create Tables:
    ```sh
    python createtable.py
    ```
### Start Server 

5. Start Server:
    ```sh
    python server.py
    ```
### TEST 

6. Start TEST:
    ```sh
    pytest
    ```
